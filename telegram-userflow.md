# Telegram UX / User Flow

## Главное меню

После команды `/start` пользователю отображается постоянная клавиатура:

1. Создать рецепт
2. Найти рецепт
3. Свободный ввод
4. Помощь

---

## 1. Создать рецепт

После нажатия:

```text
Отправьте:

• текст рецепта
• ссылку на сайт с рецептом
• ссылку на YouTube-видео
```

После получения:

Источник
→ Парсинг
→ AI-нормализация
→ Перевод на русский (при необходимости)
→ Сохранение

Пользователю показывается карточка рецепта cо всей записанной информацией из БД:

- id [обязательно]
- title [обязательно]
- description
- ingredients (JSONB) [обязательно]
- steps (JSONB) [обязательно]
- cuisine
- difficulty
- meal_type
- time_minutes

↓

Кнопки:

- Редактировать
- Сохранить

---

## 2. Найти рецепт

После нажатия:

```text
Введите запрос:

Например:

Что приготовить из курицы?
Быстрый ужин
Азиатская кухня
Без мяса
Название блюда
```

После ввода:

Запрос
→ Embedding
→ Vector Search
→ LLM Reranking

Пользователю выводятся первые 10 результатов.

Пример:

1. Пад Тай
2. Курица терияки
3. Том Ям
...
10. Карри

Каждый результат отображается отдельной Inline-кнопкой.

---

### Страница рецепта

При выборе рецепта показывается:

Название

Описание

Время приготовления

Сложность

Ингредиенты

Кнопки:

- Показать шаги
- Редактировать
- Назад к результатам
- Главное меню

---

### Показать шаги

Пошаговый рецепт.

Кнопки:

- Редактировать
- Назад
- Главное меню

---

## 3. Свободный ввод

Режим общения с AI.

Пользователь может писать произвольные запросы.

Примеры:

Добавь этот рецепт

Найди что-нибудь из курицы и картошки

Покажи последние рецепты

Измени последний рецепт

Сделай рецепт менее острым

Система определяет интент через LLM.

Поддерживаемые интенты:

- ADD_RECIPE
- SEARCH_RECIPE
- SHOW_RECIPE
- EDIT_RECIPE
- LIST_RECIPES
- HELP

Если интент не определён:

```text
Не понял запрос.

Попробуйте сформулировать иначе.
```

---

## 4. Помощь

Показывает краткую справку.

Пример:

Создать рецепт — добавить рецепт из текста, сайта или YouTube.

Найти рецепт — поиск по сохранённой базе рецептов.

Свободный ввод — общение с AI естественным языком.

Примеры запросов:

• Что приготовить из курицы?
• Найди быстрый ужин
• Замени курицу на индейку
• Покажи мои азиатские рецепты

Кнопка:

- Главное меню
